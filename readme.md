# AvProxy

## Simple IPTV proxy, small, faster, compact

- Available protocols are HTTP(S)/HLS(S)(AES-128)/UDP/RTP for inputs and outputs
- Basic/Digest Auth for ingress url
- MPTS demux for all of are input protocols
